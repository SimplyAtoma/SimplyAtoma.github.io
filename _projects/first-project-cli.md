This project was my first major Ruby build at Flatiron School.

I created a command-line application that scrapes movie data and presents it in a clean, usable format. The main challenge was learning how to structure a gem and wire the environment so everything executed correctly.

I also learned how to inspect a page, identify selectors, and parse repeated data into arrays so users could browse multiple results.
